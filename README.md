# 📸 Southdreams Photo Sorter v1.0
Organize your messy photos into Year/Month folders automatically.

## 🚀 Quick Start
1. Install Python (python.org)
2. Run: pip install Pillow
3. Put photos in 'messy_photos' folder.
4. Run: python3 photo_sorter.py
