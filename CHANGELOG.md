# Changelog — SouthDreams Photo Sorter

## [2.0.0] — Major Upgrade

### Added
- Sorts into Year / Month / File Type folder structure
- EXIF date reading (uses actual photo taken date, not file date)
- File renaming to date format (2024-03-15_filename.jpg)
- Duplicate detection with interactive user prompt
- RAW camera file support (CR2, CR3, NEF, ARW, DNG, ORF, RW2, PEF, RAF)
- Screenshot detection by filename
- Full sort report saved as sort_report.txt
- Safe preview mode on by default
- Android/Termux launcher (run-android.sh)
- Windows launcher (run-windows.bat)
- Auto-installs Pillow if not present

## [1.0.0] — Initial Release

### Added
- Basic Year/Month sorting
- Windows and Android launchers
